# HealthCheck AI 🩺

HealthCheck AI is a mini symptom checker application built with **FastAPI** and **Scikit-Learn**. It uses a Random Forest classifier trained on a synthetic dataset of 40+ diseases and 130+ symptoms to provide real-time predictions.

## Features
- **AI-Powered Diagnostics**: Predicts possible diseases with confidence percentages.
- **Premium UI**: Modern glassmorphism design with a smooth user experience.
- **Health Tips**: Provides basic guidance and severity levels for each predicted condition.
- **Real-time Search**: Quick symptom selection with intelligent suggestions.

## Tech Stack
- **Backend**: Python, FastAPI, Scikit-Learn, Pandas.
- **Frontend**: HTML5, Vanilla CSS (Modern Aesthetics), Vanilla Javascript.

## How to Run
1. Ensure you have Python installed.
2. Install dependencies:
   ```bash
   pip install fastapi uvicorn scikit-learn pandas numpy
   ```
3. Run the startup script:
   ```powershell
   ./run.ps1
   ```
4. Open the app at `http://localhost:3000`.
